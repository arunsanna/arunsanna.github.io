MicroEKS

DevOps Tool
View on GitHub

Lightweight tool for creating local Kubernetes environments on macOS that mirror AWS EKS. Uses MicroK8s + Multipass to provide a production-like K8s experience for local development.

Key Features

  • One-command setup for local Kubernetes clusters matching AWS EKS configuration
  • Distributed via Homebrew tap for easy installation on macOS
  • Leverages MicroK8s and Multipass for lightweight virtualization
  • Enables local testing of production Kubernetes configurations before deployment
